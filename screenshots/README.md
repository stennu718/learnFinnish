# Screenshots & GIFs

This directory stores visual assets for documentation (README, GitHub, etc.).

## What to capture

### Screenshots
1. **Flashcard view** — Main learning interface showing a Finnish word with hint/reveal
2. **Grammar view** — Grammar engine UI showing case transformations
3. **Progress dashboard** — User progress, streaks, and XP stats
4. **Mobile view** — Mobile app flashcard experience
5. **Auth screens** — Login and registration pages

### GIFs (short, <10s)
1. **Flashcard flip** — Tapping to reveal the answer
2. **Streak animation** — XP gain and streak counter incrementing
3. **Keyboard navigation** — Using arrow keys / space to navigate cards

## File naming convention

```
screenshots/
  01-flashcard-view.png
  02-grammar-view.png
  03-progress-dashboard.png
  04-mobile-view.png
  05-auth-screens.png
  demo-flashcard-flip.gif
  demo-streak-animation.gif
  demo-keyboard-nav.gif
```

## Tips

- Use a consistent browser window size (e.g., 1440x900 for desktop)
- Clear personal data before capturing (use a test account)
- For GIFs: keep under 5MB for fast loading on GitHub
- Preferred tools: `scrot` / `flameshot` (Linux), `gyazo` (GIF), or browser devtools
